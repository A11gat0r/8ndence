How it works

This tool performs simple metadata similarity checks to flag potentially suspicious apps. It compares three things:

  • App name similarity â€” compares the brand text to the application name.
  • Package id similarity â€” compares a generated package id (e.g., com.brand) to the actual package identifier.
  • Publisher similarity â€” compares the brand to the publisher name reported on the store.

The tool averages the three similarity scores and converts that into a risk score using risk = 100 - averageSimilarity. Higher risk means lower similarity to the given brand and therefore more suspicious.

This is a demo algorithm for educational purposes and should not be used as the only signal for critical decisions.
